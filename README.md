# 习题库

---

采用J2EE开发的答题系统，使用了七牛云存储技术，itext、ECharts等插件，前端使用mui框架。包括答题（选择、判断、填空、简答、计算）、系统自动判题（选择、判断、填空）、管理员判题（简答、计算）、查看详情、打印pdf、查看统计情况等功能。
